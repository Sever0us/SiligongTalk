What is distributed Computing

“A distributed system is a model in which networked computers interact with each other in order to achieve a common goal.” ~ G. Coulouris
“If that goal isn't world domination, you are wasting your time.”

~ Me

How does distributed computing work?

  • We connect multiple machines together in a network

  • Machines share information via messages

  • A machine can take a message and use it to perform a job

  • The job result is reported via another message

But what does the network look like?

Client-server
Tiered
Peer-to-peer

When should we use distributed computing?

  • If you need to deploy code in different physical localtions

  • Large ammounts of isolated tasks

  • Cost effective performance

  • Failure redundancy

Embarrassingly parallel tasks

  • Serving static web files to multiple users

  • Computer rendering

  • Physical simulations (climate modelling, cosmology)

  • GPU deployed code

Unsuitable tasks

  • Inherrently sequential tasks (pregnancy)

  • Communication intensive tasks (matrix multiplication)
